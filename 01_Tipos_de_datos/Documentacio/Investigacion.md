# Investigacion

## ¿Que es un tipo de dato?
Son datos basicos indivisibles que almacenan valores true/false ayudan a construir estructuras mas complejas. De los cuales estos son: int, float, double, char.

## ¿Cuales son los datos no primitivos?
Son los datos de referencia o de estructura compleja que almacena direcciones de memoria a los datos reales. Como: bool, null, string, struc.

## Importancia del lenguaje C

Es importante el lenguaje c ya que se aplica directa o indirectamente en muchas areas por ejemplo en la electronica y la robotica por su eficiencia y rendimiento al acceder directamente al hardware.\
Sienta las bases para el desarrollo de software de bajo nivel y fundamento a otros lenguajes como Python, PHP, C++. Y es el lenguaje mas practico para programar microcontroladores y sistemas embebidos.
